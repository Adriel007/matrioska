# Responsive Landing Page

Create a responsive landing page for a fictional SaaS product called "CloudFlow" (a cloud workflow automation tool).

## Requirements

- HTML file with semantic structure (header, hero, features section, footer)
- CSS file (or inline styles) that makes it responsive (mobile-friendly)
- Hero section with: headline, subheadline, CTA button that links to "#signup"
- Features section with 3 feature cards (icon, title, description)
- Responsive: use CSS media queries or flexbox/grid to adapt layout
- Color scheme: professional (blues/whites/greys)
- Footer with copyright and placeholder links

## Output Files

- `index.html`: Main HTML page
- `styles.css`: CSS styles
